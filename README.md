# OurOwn_Shell
## 小组成员：王新源、蒲雯
在Linux环境下，结合 C++标准库和 Linux 的系统调用，完成的一个Shell程序
该程序的功能如下：
1. shell 程序能够提供命令的输入，执行并显示执行结果的功能。
2. shell 程序能够提供可 shell 编程的功能，能够执行简单的 shell 脚本。
3. shell 程序能够提供 I/O 重定向和管道的功能。
